# Feature: Folder Browser in Chat Tab

## Summary
Users can visually browse and select directories for working directory instead of typing paths manually.

## Acceptance Criteria
- [ ] Browse button next to CWD input
- [ ] Modal shows folder list from backend
- [ ] Can navigate into folders
- [ ] Windows shows drives, Unix shows root
- [ ] Select button sets CWD
- [ ] Cancel closes without changes
- [ ] Handles permission errors gracefully
