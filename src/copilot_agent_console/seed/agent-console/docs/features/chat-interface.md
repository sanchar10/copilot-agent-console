# Feature: Chat Interface

## Summary
Users can have natural language conversations with GitHub Copilot through a ChatGPT-like interface with real-time streaming responses.

## Acceptance Criteria
- [x] Text input box with send button
- [x] Messages display with role-based styling (user vs assistant)
- [x] Responses stream in real-time (not all at once)
- [x] Tool execution steps visible during response (collapsible Steps block with max-height scrolling)
- [x] Chat history persists across page refreshes
- [x] Scroll to the bottom of the chat
- [x] Keyboard shortcuts: Enter to send, Shift+Enter for newline
