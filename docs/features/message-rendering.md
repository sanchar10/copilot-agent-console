# Feature: Message Rendering

## Summary
Assistant messages are rendered with rich formatting including Markdown, syntax-highlighted code blocks, tables, and other elements.

## Acceptance Criteria
- [x] Markdown parsed in assistant messages
- [x] Code blocks have syntax highlighting (language auto-detected from fence)
- [x] Tables render with proper styling (GFM tables)
- [x] Links are clickable and open externally
- [x] Inline code has distinct styling
- [x] Lists, blockquotes, headings styled appropriately
- [x] User messages remain plain text
